HTML/CSS

На цій сторінці ви знайдете основну інформацію пов'язану з HTML і CSS.

Що ж таке HTML?

HTML (Hyper text markup language) – це мова для розмітки тексту, що дозволяє описувати веб-документи. HTML документи формуються за допомогою HTML тегів. Кожен HTML тег описує певну частину контенту або метаінформації документа.

Де використовується HTML?

HTML документ описується в текстовому вигляді, але якщо створити файл із розширенням .html (або .htm), внести у нього HTML текст і відкрити у браузері, то браузер відобразить його у графічні елементи, що описані тегами. Саме це перетворення розміченого тексту у графічний документ і відбувається щоразу при завантаженні сторінки у веб-браузері. Також поширеним застосуванням HTML є: Веб-розробка: HTML використовується для створення структури і змісту веб-сторінок. Він визначає заголовки, параграфи, списки, таблиці, посилання, зображення та інші елементи, які відображаються в браузері. Мобільна розробка: HTML разом з CSS і JavaScript використовується для створення мобільних веб-сторінок та веб-додатків, які можуть відображатися на різних пристроях і розмірах екранів. Електронна пошта: HTML використовується для створення структурованих та стилізованих електронних листів, які можуть містити текст, зображення, посилання та інші елементи.

Як щодо прикладів програмного коду на HTML?

Розглянемо приклад з Hello world на HTML.


            
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
     <title>Hello World Example</title>
</head>
<body>

<h1>Hello, World!</h1>

<script>
    // JavaScript код для виведення "Hello, World!" в консоль браузера
    console.log("Hello, World!");
</script>

</body>
</html>

Сенс показувати якісь великі проекти на чистому HTML сенсу мало, оскільки зараз HTML використовується разом з CSS та JavaScript. Тому давайте розберемось з CSS.

Що таке CSS?

CSS - це спеціальна мова стилю сторінок, що використовується для опису їхнього зовнішнього вигляду. Самі ж сторінки написані мовами розмітки даних. CSS є основною технологією всесвітньої павутини, поряд із HTML та JavaScript.

Як виглядає код на CSS? Для прикладу буде використано пару стрічок, які використовувались при розробці цього сайту і цієї сторінки.


            
h1 {
    word-wrap: break-word;
    overflow-wrap: break-word;
    white-space: normal;
    line-height: 1.2;
}

.text_block{
    margin: 10px 50px;
    box-shadow: 0 0 10px rgba(0, 0, 0, 0.1);
    background-color: #333333;
    border-radius: 10px;
    padding: 2px;
    margin-bottom: 20px;
}

.text .main_text{
    font-size: 62px;
}

body{
    background: #212121;
    font-size: 22px;
    line-height: 32px;
    color: #fff;
    word-wrap: break-word;
    font-family: 'Lucida Sans', 'Lucida Sans Regular', 'Lucida Grande', 'Lucida Sans Unicode', Geneva, Verdana, sans-serif;
}
                
a {
    color: #fff;
}

Тобто, як ми бачимо, код який можна було б використати разом з HTML можна виокримити в окремий файл .css і він буде працювати на всі сторінки, які підключені до цього файлу .css. Тобто, якщо раніше стилі потрібно було прописувати для кожної сторінки окремо, то тепер ці стилі можна винести в окремий .css файл і підключати для прикладу до сторінок одного проекту. Для підключення використовується наступний код:


                
<link rel="stylesheet" href="css/style.css">
