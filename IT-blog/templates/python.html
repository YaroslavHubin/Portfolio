Python

На цій сторінці ви знайдете основну інформацію пов'язану з такою мовою програмування, як Python.

Що ж таке Python?

Python - інтерпретована об'єктно-орієнтована мова програмування високого рівня із суворою динамічною типізацією. Розроблена в 1990 році Гвідо ван Россумом. Структури даних високого рівня разом із динамічною семантикою та динамічним зв'язуванням роблять її привабливою для швидкої розробки програм, а також як засіб поєднування наявних компонентів. Python підтримує модулі та пакети модулів, що сприяє модульності та повторному використанню коду. Інтерпретатор Python та стандартні бібліотеки доступні як у скомпільованій, так і у вихідній формі на всіх основних платформах. В мові програмування Python підтримується кілька парадигм програмування, зокрема: об'єктно-орієнтована, процедурна, аспектно-орієнтована та функціональна.

Де використовується Python?

Python використовуються у багатьох сферах, таких як: ігрова індустрія, веб-дизайн, прикладне програмування тощо. Але Python, окрім того що це дуже проста мова, котра не потребує приділення великої уваги до синтаксису, тому що він там дуже простий (синтаксис), він ще є повільною мовою, що дає свої певні обмеження. Тобто незваєаючи на те, що він має дуже багато бібліотек, які можна використати для створення красивих і водночас не дуже складних речей, ці речі мають бути не надто складними для обчислень , оскільки у Python`і з цим є проблеми, через те що він є інтерпретованою мовою.

Як щодо прикладів програмного коду на Python?

Розглянемо приклад з Hello world на Python.


            
def hello_world():
    print("Hello, world!")
hello_world()

Ще також можна навести приклади використання циклів та умов. Ось приклад використання умов на Python:


            
def hello_world():
    print("Hello, world!")
n=2
if n != 1:
    hello_world()
else:
    print("Error, n=1")

Ось приклад використання циклів на Python:


            
def hello_world():
    print("Hello, world!")
n=1
for i in range(1,10);
    if n == 10:
        hello_world()
    else:
        print("Error, n="+i)
        n+=1

Окрім звичайних програм, Python також гарно підходить для тренування моделей машинного навчання. Ось приклад з реального проетку, разом з використанням ML (Machine learning) для передбачення погоди:


            
import pandas as pd
import numpy as np
from datetime import datetime, timedelta
from sklearn.linear_model import LinearRegression
import matplotlib.pyplot as plt
                
data = pd.read_excel('погода.xlsx')
                
data['timestamp'] = pd.to_datetime(data['timestamp'], format='%Y%m%dT%H%M')
                
data['day_of_year'] = data['timestamp'].dt.dayofyear
data['hour'] = data['timestamp'].dt.hour
                
X = data[['day_of_year', 'hour']].values  # день року та година
y = data['Basel Temperature [2 m elevation corrected]'].values  # Температура
                
model = LinearRegression()
model.fit(X, y)
                
last_timestamp = data['timestamp'].max()
future_dates = [last_timestamp + timedelta(hours=i) for i in range(1, 24*365*20+1)]
                
future_day_of_year = [date.dayofyear for date in future_dates]
future_hour = [date.hour for date in future_dates]
                
future_predictions = model.predict(np.column_stack((future_day_of_year, future_hour)))
                
for i in range(20):
    start_idx = i * 24*365
    end_idx = (i + 1) * 24*365
    year_predictions = future_predictions[start_idx:end_idx]
    plt.plot(future_dates[start_idx:end_idx], year_predictions, label=f'Year {i+1}')
                
plt.title('Temperature Predictions for Next 20 Years')
plt.xlabel('Date')
plt.ylabel('Temperature')
plt.legend()
plt.show()

Тут наведено приклад розбиття цілого датасету на окремі колонки часу(рік, місяць, день, година). Також задано модель лінійної регресії для передбачення погоди на наступні 20 років. Також ось візуальне відображення розподілу даних і тренування моделі.
